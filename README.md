# Plat-Mayhem

A high-performance Python platformer featuring a professional-grade procedural Endless Mode with dynamic difficulty scaling.

## 🚀 Features
- **Procedural Generation**: Infinite level chunks using a modular `ChunkGenerator`.
- **Dynamic Difficulty**: Real-time scaling based on survival time and player score.
- **Modular Architecture**: Independent systems for spawning, loot distribution, and enemy AI.
- **Optimized Performance**: Stable 60 FPS with efficient memory management and off-screen cleanup.

## 🛠️ Installation & Play
1. Ensure you have `pygame-ce` installed: `pip install pygame-ce`
2. Run the game: `python main.py`
3. Select **Stage 3** to experience the Endless Mode.

Developed with a focus on SOLID principles and clean code.
